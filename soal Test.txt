Screen 1 :
Menampilkan hasil list input
Screen 2 :
Menampilkan form input, dengan isian:
Nama
Alamat
Tanggal Lahir (tidak boleh diketik)
Tinggi (cm) (input berupa angka)
Berat (kg) (input berupa angka)
Foto (input bisa dari kamera maupun gallery)
Submit

Menampilkan detail dari data, dimana akan ada aksi dari user berupa edit maupun hapus data

detik, button submit akan disable.
Format Timer mm:ss
estimasi waktu 3 - 4 jam
